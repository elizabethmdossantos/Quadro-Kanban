# Quadro Kanban Simples

Um quadro **Kanban** minimalista e funcional implementado com **HTML**, **CSS** e **JavaScript** puro, utilizando **armazenamento local** (localStorage) para persistir as tarefas.

## Visão Geral do Projeto

Este projeto implementa um sistema de gerenciamento de tarefas visual, seguindo a metodologia Kanban com as colunas:

* **A Fazer** (To Do)
* **Em Andamento** (Doing)
* **Concluído** (Done)

As tarefas podem ser adicionadas, excluídas e arrastadas (Drag-and-Drop) entre as colunas para refletir o seu estado atual.

## Funcionalidades

* **Adicionar** novas tarefas.
* **Excluir** tarefas individualmente.
* **Limpar** todas as tarefas com um único botão.
* **Editar** o conteúdo da tarefa com um clique no ícone de lápis ou um duplo clique no texto.
* **Arrastar e Soltar** (Drag-and-Drop) para mover tarefas entre as colunas.
* **Persistência de Dados**: As tarefas são salvas no **localStorage** do navegador, mantendo o estado do quadro entre as sessões.
* **Responsivo**: Layout adaptado para visualização em dispositivos móveis e desktops.

## Tecnologias Utilizadas

* **HTML5** (Estrutura)
* **CSS3** (Estilização e Responsividade)
* **JavaScript ES6+** (Lógica do Kanban, Manipulação do DOM e Persistência de Dados)
